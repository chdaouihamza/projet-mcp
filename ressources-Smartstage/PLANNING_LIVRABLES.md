# Planning & Livrables — SmartStage

## Plan de Développement (4 mois)

| Mois | Phase | Travaux |
|------|-------|---------|
| M1 | Analyse & Conception | Recueil des besoins, cahier des charges, diagrammes UML, maquettes UI/UX (Figma), initialisation projets Spring Boot et Angular |
| M2 | Développement Backend | Authentification JWT, gestion des utilisateurs et rôles, API REST pour profils, sujets et candidatures, module de tests, intégration PostgreSQL |
| M3 | Développement Frontend | Interfaces Angular (dashboard RH, espace employé, espace stagiaire), intégration APIs, module de test avec chronomètre, notifications WebSocket |
| M4 | Finalisation & Déploiement | Module d'archivage, tests unitaires, correction des bugs, conteneurisation Docker, rapport final, soutenance |

## Livrables Attendus

- Cahier des charges fonctionnel et technique
- Maquettes UI/UX (Figma)
- Diagrammes UML complets (classes, séquences, activité, déploiement)
- Application web déployée (Angular + Spring Boot)
- Base de données PostgreSQL modélisée et peuplée
- Infrastructure Docker (docker-compose)
- Rapport PFA complet (80–120 pages)
- Présentation de soutenance
- Vidéo démo de l'application (5–10 minutes)

## Compétences Mobilisées

- Développement backend avec Java et Spring Boot (REST, JPA, Security)
- Développement frontend avec Angular 17 (composants, services, guards, RxJS)
- Conception de bases de données relationnelles (PostgreSQL, modélisation UML)
- Gestion de l'authentification et de la sécurité (JWT, BCrypt, RBAC)
- Communication en temps réel (WebSocket, STOMP)
- Stockage objet pour les médias (MinIO / AWS S3)
- Conteneurisation et déploiement (Docker, Docker Compose)
- Méthodologie de développement agile (Scrum simplifié)

## Conclusion

Le projet SmartStage représente une opportunité idéale de Projet de Fin d'Année, alliant une problématique concrète et réelle, un scope fonctionnel riche et bien délimité, et une stack technique moderne et professionnalisante.

Sa portée fonctionnelle (gestion des candidatures, tests automatisés, vote collaboratif, archivage enrichi) offre suffisamment de profondeur technique pour produire un rapport de qualité et démontrer une maîtrise avancée du développement web full-stack avec Java Spring Boot et Angular.

**Résumé Exécutif** — SmartStage est une plateforme web full-stack (Spring Boot + Angular) permettant à une entreprise de gérer l'intégralité du cycle de vie des stages PFA : de la candidature à l'archivage, en passant par l'évaluation technique automatisée et la sélection collaborative des sujets.
